# 동영상 보고서 작성 및 촬영 가이드

본 문서는 개발 과정이나 결과물을 동영상으로 기록하고 보고하는 방법을 설명합니다.

---

## 1. Jetson 화면 녹화 방법

Jetson(Ubuntu) 환경에서 가장 사용하기 편한 녹화 도구들을 소개합니다.

### 방법 A: SimpleScreenRecorder (추천)
가장 안정적이고 설정이 쉽습니다.
1.  **설치**: 터미널에서 `sudo apt install simplescreenrecorder`
2.  **실행**: 메뉴에서 'SimpleScreenRecorder' 검색 후 실행.
3.  **설정**: 전 영역 녹화 또는 특정 창 녹화를 선택할 수 있습니다.
4.  **저장**: 녹화 종료 후 `.mp4` 또는 `.mkv` 파일로 저장됩니다.

### 방법 B: Kazam
매우 심플하고 가볍습니다.
1.  **설치**: `sudo apt install kazam`
2.  **기능**: 단축키로 녹화 시작/종료가 가능합니다.

---

## 2. 동영상 보고서 구성 팁

효과적인 기술 보고를 위해 동영상에 다음 내용을 포함하는 것이 좋습니다.

1.  **환경 확인**: 통신 인터페이스(`eno1`) 설정 확인 과정.
2.  **빌드 과정**: `~/soem_lib_build`에서 `./rebuild_so.sh` 실행 장면.
3.  **Qt 실행**: Qt Creator에서 `Run` 버튼 클릭 후 콘솔에 로그가 찍히는 장면.
4.  **슬레이브 확인**: 연결된 EtherCAT 슬레이브 개수가 출력되는지 확인.

---

## 3. 결과물 공유 방법

녹화된 파일은 용량이 클 수 있으므로 다음과 같이 관리하세요.
*   **파일 압축**: `handbrake` 같은 도구로 용량을 줄일 수 있습니다.
*   **문서 포함**: 본 문서(`walkthrough.md`) 등에 녹화 파일의 경로를 링크로 걸어 두시면 나중에 확인하기 좋습니다.
